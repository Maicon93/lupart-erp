# commits.md

- Seguir **Conventional Commits**
- Formato: `tipo(escopo): descrição`
- Tipos: `feat`, `fix`, `refactor`, `chore`, `docs`, `style`, `test`
- Escopo: nome do módulo em inglês (`products`, `sales`, `auth`, `stock-entry`, `service-orders`)
  - Escopos globais: `config`, `db`, `i18n`, `deps`
- Descrição em inglês, curta e no imperativo (`add`, `fix`, `update`, `remove`)
- Exemplos:
  - `feat(products): add product listing and creation`
  - `fix(sales): correct installment calculation`
  - `refactor(auth): simplify token refresh logic`
  - `chore(deps): update quasar to 2.x`
  - `docs(stock-entry): update skill documentation`
